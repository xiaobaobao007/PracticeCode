1：一个抽象类并无什么特殊操作，但是继承和实现了接口：用于整合多个抽象类于一体，也更方便于再被继承
2：一横行的划线格子包含多行数据：用于比较。

⬛⬛⬛⬛⬛⬛一：抽象父类等基础类⬛⬛⬛⬛⬛⬛
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Iterator                                |接口：应用在Iterable接口，即只要实现Iterable接口，即可使用迭代器。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Collection                              |接口，继承Iterable：定义常见集合的方法。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
AbstractCollection                      |抽象类，实现Collection接口：实现Collection方法。
List                                    |接口类，实现Collection接口：定义常见“顺序列表”集合的方法。
Queue                                   |接口类，实现Collection接口：定义常见“链表列表”集合的方法(单项队列)。
Set                                     |接口类，实现Collection接口：定义常见“集和映射”集合的方法。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Deque                                   |接口类，继承Queue：定义“双向队列”集合的方法。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
AbstractList                            |抽象类，继承AbstractCollection，实现List接口：快速开发“随机访问”的集合类。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Map                                     |接口类，定义映射集合的常用方法。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
AbstractMap                             |抽象类，实现Map接口：快速开发“映射”的集合类。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
AbstractQueue                           |抽象类，继承AbstractCollection，实现Queue接口：用于快速开发“链表”的集合类。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
RandomAccess                            |接口类：无任何实现，只是一个标示，可以根据索引操作，比迭代器速度更快。
AbstractSequentialList                  |抽象类，继承AbstractList：为链表结构提供索引操作。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
AbstractSet                             |抽象类，继承AbstractCollection，实现Set接口。
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
